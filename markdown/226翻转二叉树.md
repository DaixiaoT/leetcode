# 226翻转二叉树



### 递归法

```cpp
TreeNode* invertTree(TreeNode* root) {
        if(root==nullptr){
            return nullptr;
        }
        TreeNode *temp;
        temp=root->left;
        root->left = root->right;
        root->right=temp;
        invertTree(root->left);
        invertTree(root->right);
        return root;
    }
```



### 辅助队列法

```cpp
TreeNode *invertTree(TreeNode *root)
    {
        queue <TreeNode*> q;
        if (root == nullptr)
        {
            return nullptr;
        }
        q.push(root);
        while (!q.empty())
        {
            TreeNode *temp;
            temp = q.front();
            q.pop();
            swap(temp->left,temp->right);
            if (temp->left != nullptr)
            {
                q.push(temp->left);
            }
            if (temp->right != nullptr)
            {
                q.push(temp->right);
            }
        }

        return root;
    }
```
